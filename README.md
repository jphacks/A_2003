## QOL×Tech
### 背景(製品開発のきっかけ、課題等）


朝起きるのがつらい。しかし、特に1限がある日は朝早く起きる必要がある。そこで、朝早くなおかつ快適に起きるための便利なグッズがあればいいと思った。そこで、朝の設定した時間になったら自動でカーテンを開ける装置を開発した。

### 製品説明（具体的な製品の説明）

　人体の仕組みとして、朝日を5分以上浴びることでメラトニンが抑制されて目覚めが良くなります。朝に自動でカーテンが開くことで、利用者は勝手に外から届く朝日を浴びることになり、快適に目覚めることが出来ます。
